
**Pasos para lanzar el proyecto**
- >git clone https://github.com/clintonkenedy/restech.git
- copiar el .env.example en -env
- confirurar el .env
- crear la base de datos restech
- >composer install
- >php artisan migrate --seed
- >php artisan key:generate
- >npm install
- >npm run dev
